Preguntas.

Lab 1

  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    El Internet son todas las computadoras y la conexión que tienen, mientras que la WWW es la red de información que se puede acceder a través de Internet (Redacción, 2019).


  • ¿Cuáles son las partes de un URL?

    (Collado, 2025)

    1. Protocolo.
    2. Subdominio.
    3. Dominio.
    4. TLD.
    5. Carpeta o subcarpeta.
    6. Página.
    7. Etiqueta.

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    1. GET: Es utilizado para obtener información de un recurso.
    2. HEAD: Es utilizado para obtener información de un recurso sin obtener el cuerpo del recurso.
    3. POST: Es utilizado para enviar un nuevo recurso, causando un cambio de estado.
    4. PUT: Es utilizado para actualizar un recurso existente.
    5. PATCH: Es utilizado para actualizar un recurso existente de manera parcial.
    6. DELETE: Es utilizado para eliminar un recurso existente.

  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

    Se usa POST debido a que son datos sensibles entonces se colocan en el cuerpo y mediante POST no se envían en la red.


  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

    Se utiliza GET ya que se está solicitando información de un recurso.


  • Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

    El código 200 significa que la solicitud se ha realizado correctamente.


  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

    No, ya que el desarrollador no controla que URL fue ingresada por el usuario entonces no es su responsabilidad.


  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

    Sí, porque el desarrollador controla el código de la página web.


  • ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

    Significa que el atributo ya no se utiliza pero hay varios que con los que los puedes sustituir. Algunos elementos de HTML 4 que en HTML5 estén desaprobados son: applet, acronym, bgsound, dir, hgroup, isindex, basefont, big, center, font, menu, etc (Estas Son Las Etiquetas Obsoletas En HTML5: ¡Actualízate! | Tokio, 2022).


  • ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

    Soporte para video, audio, canvas, svg geolocalización, web storage


  • ¿Qué componentes de estructura y estilo tiene una tabla?

    Una tabla registros (filas) y campos (columnas).


  • ¿Cuáles son los principales controles de una forma HTML5?

    Los principales controles en HTML5 son: input, select, textarea, button, checkbox, radio, file, password, submit, reset, etc. (Controles De Formulario Originales - Aprende Desarrollo Web | MDN, n.d.)

    • Sobre el ciclo de vida y desarrollo de los sistemas de información:

  • ¿Cuál es el ciclo de vida de los sistemas de información? ¿Cuál es el ciclo de desarrollo de sistemas de información?

    El ciclo de vida de los sistemas de información es el conjunto de fases que un sistema de información atraviesa desde su concepción hasta su desmantelamiento.
    Mientras que el ciclo de desarrollo es más acerca desde que se le plantea un problema hasta que se le implementa la solución.

    • Lab 3

  • Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

    Es algo que no se recomienda usar debido a que es un último recurso y puede afectar la estructura de tu CSS.


  • Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

    Debido a que puede interferir con la facilidad para leer el texto así como los elementos en pantalla.


  • Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

    Aunque varia de acuerdo a que elemento se esté tratando, en general se recomienda usar px ya que es la unidad más común y fácil de usar, sin conversiones.


  • ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

    Debido a que los saltos de línea y espacios generan un mayor peso y tiempo de carga en el envío de la página.

    • Lab 4

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Los dos son lenguajes de programación orientados a objetos. Java es compilado, mientras JavaScript es interpretado. Java se utiliza para crear aplicaciones de escritorio y móviles, mientras JavaScript se utiliza para crear aplicaciones web.


  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    El objeto Date tiene los siguientes métodos:
    getDate, getDay, getFullYear, getHours, getMinutes, getMonth, getSeconds, getTime, etc.


  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    Los arreglos tienen los siguientes métodos:
    push, pop, shift, unshift, splice, sort, reverse, etc.


  • ¿Cómo se declara una variable con alcance local dentro de una función?

    Se declara con la palabra reservada let o const.


  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Puede ser peligroso debido a que pueden ser modificadas desde cualquier parte del código. Alguien puede acceder a las variables desde la consola de desarrollador.
